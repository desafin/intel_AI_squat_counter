# Project name


스쿼트 카운터
1.웹캠으로 실시간으로 영상을 받아와 포즈 추정 AI를 사용하여 스쿼트를 할때마다 카운트를하고
2.실시간으로 화면의 백그라운드를 제거해 사용자의 몸만 보이게 한다
3.스쿼트의 갯수를 화면에 표시한다.
